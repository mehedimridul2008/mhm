# Mehedi Hasan Mridul - Personal Website

This is the official personal website of Mehedi Hasan Mridul, a student at Notre Dame College Mymensingh, Bangladesh.

## Features

- Responsive design for all devices
- Bengali language support
- Interactive timeline of life events
- Contact information and social media links
- Visitor counter
- Real-time date and time display (including Islamic date)

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome icons
- Google Fonts (Hind Siliguri)

## How to Use

1. Clone this repository
2. Open `index.html` in your browser
3. Explore the website sections using the navigation menu
